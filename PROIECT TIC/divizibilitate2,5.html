        Un numar n este prim daca si numai daca acesta are doar 2 divizori, acestia fiind 1 si n. Folosind secventa aratata anterior, la Multimea divizorilor, un numar n este prim daca si numai daca variabila nr are valoarea 0. Dar pentru eficienta maxima se poate folosi o variabila de tipul boolean(ok) care isi schimba valoarea de la true la false cand se gaseste un divizor. Astfel, structura repetitive se opreste mai repede si nu mai continua in cautarea divizorilor. Acest lucru este realizat astfel:

        #include < iostream >

        int main(){
        int n,d;
        bool ok=true;
        cin >>n;
        if(n!=1){
           for(d=2;(d*d<=n)&(ok==true);d++)
              if(n%d==0)ok=false;}
           else ok=false;
           if(ok==true)
              cout<< "nr prim"; else cout<< "nu e nr prim";
